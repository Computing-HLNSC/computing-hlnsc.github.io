Step 1: Enabling HyperV as feature

1. Navigate to the Control Panel. Select Start, then search for Control Panel to open the application.


2. Select Programs, then Programs and Features.


3. Select Turn Windows Features on or off.


4. Select Hyper-V and then select OK.


General Information

You only need to complete this step if Hyper-V Manager is not present on your local machine/computer.


Important Information

HyperV is a feature which is only available on Pro, Enterprise, and Education versions of Windows.

Step 2: Deciding where to host the Virtual Machine

1. Open Hyper-V Manager.


2. Right click Hyper-V Manager in the left column


3. Select `Connect to Server...`


4. Select where you want to host the virtual machine.


General Information

Virtual machines can be created either on the local machine (the computer you are using) or on a remote machine (another computer).


The machine which has the virtual machine stored on it, is called the host.


Important Information

If you are connecting to another host, such as a server on a network. You may find you need administrator account privileges. Simply run Hyper-V Manager as admin.

Step 3: Creating the virtual machine

1. Select the host in the left column.


2. Select `New` ‚Üí `Virtual Machine` in the right column.

Step 4: Configuring (Specify Name and Location )

1. Type the name of your virtual machine


2. Press `Next`


Important Information

You should follow a similar naming convention as detailed below.

{CourseLevel}_{Firstname}_{Lastname}_{VirtualMachineNumber}

Example: L4_Harry_Potter_0

Step 5: Configuring (Specify Generation)

1. Select the generation of virtual machine you wish to create


2. Press `Next`


General Information

Generation 1: Use for legacy operating systems or when older hardware emulation (BIOS, IDE, legacy network adapters) is required.


Generation 2: Use for modern operating systems supporting UEFI, Secure Boot, and faster performance through virtual SCSI and PXE boot improvements.

Step 6: Configuring (Assign Memory)

1. Enter the start up memory


2. Press `Next`


General Information

You need to weigh up memory needs of the virtual machine against the available resources of the host.

Step 7: Configuring (Configure Networking)

1. Select the network connection you wish the virtual machine to use.


2. Press `Next`


General Information

Virtual network switches and subsiquent connections to those switches can be setup later.

Step 8: Configuring (Configure Virtual Hard Disk)

1. Enter the size of storage you wish to allocate to the machine.


2. Press `Next`


General Information

You need to weigh up storage needs of the virtual machine against the available resources of the host.

Step 9: Configuring (Installation Options)

1. This is where you select if you are going to install the operating system now, or do it later.


2. Press `Next`


General Information

If you are connected to a remote host, then the operating system you want to install (ISO file) needs to be accessible from the host.

Step 10: Summary

1. Verify the choices and ammend anything if needed.


2. Press `Finish`
