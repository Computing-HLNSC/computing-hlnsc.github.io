TurnItInUK

TurnitinUK is an online tool used by colleges and universities to check student work for originality. It compares submitted work against a large database of books, websites, journals, and past student papers. It then produces an originality report showing any matched text so tutors can see whether sources have been properly referenced.

When submitting to TurnitinUK, students must ensure all sources are properly cited, all quotations are clearly marked, and any paraphrased content is genuinely rewritten and referenced.
The originality report highlights matching text so tutors can check that all borrowed material has been correctly acknowledged.

Logging in

To login to your account, you need to either create an account using your organisational email, or login using the link sent to you by your tutor.

Navigating TurnItInUK

Once you have logged in you can see the modules/units you have been enrolled in.

Select one of these units.

Viewing a Units assignments

Once you have selected a unit you can see the assignments associated with this unit.

This is where you can press Open and see further assignment details.

Viewing a Units assignments

This is where you can see assignment details set by the tutor and upload your submission for marking.

Reminder

Reminder that you should follow the submission guidance on the assignment brief. Generally your submission should be in a single .pdf file format.

You may want to use the sites Merge PDF Tool in order to merge different tasks and documents for your assignment.

Viewing a Units assignments

Once you have uploaded your submission you will see a preview/summary page. Select confirm at this stage when you are happy.

You generally have until the deadline to resubmit/ammend your assignment.
