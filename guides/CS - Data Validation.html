Data Validation

Explanation:
Data validation checks that data entered into a system is correct, sensible, and acceptable before the system processes it. It stops invalid, dangerous, or incorrect data from causing errors, corrupting databases, or breaking functionality. This keeps systems accurate, secure, and reliable.

Steps:
The steps on this guide go through the different types of data validation checks you can find within a code base.
